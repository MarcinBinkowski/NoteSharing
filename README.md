# Notes

An ephemeral note-sharing application.

**Live:** [acc.mbinkowski.tech](https://acc.mbinkowski.tech)

---

## Features

- Create notes with an optional **expiration time** (from 1 minute to 30 days)
- **Burn after reading** — note is deleted after the first view
- **Password protection** — protect note content with a password
- **Anonymous or authenticated** — sign in with Google to manage your notes
- View and delete your own notes from the **My Notes** page
- Note owners can bypass password protection on their own notes
